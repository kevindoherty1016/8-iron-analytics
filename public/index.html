Import Data

Upload your hole-by-hole spreadsheet data to build your dashboard.

Basic Format Requirements

Import your aggregate round totals. Your file must contain these exact headers:
Date, Course, Course Par, Score, Holes, GIR , Putts, Eagles, Birdies, Pars, Bogeys, Double Bogeys, Triple Bogeys, Penalty Strokes, Up/Down Chances, Up/Down Successes, Putter Used, Cost, Winnings, Event, Group

Tip: To batch-assign courses/tees to existing rounds, upload a CSV with Round # and either Course ID and/or Tee ID.

Download CSV Template

Detailed Format Requirements

Import granular, hole-by-hole analytics. Your file must contain these exact headers:
Date, Course, Tee ID, Hole, Par, Score, FIR, GIR, Scrambling, Putts, Putter Used, Cost, Winnings, Event, Group

Ensure you have exactly one row per hole (e.g., 18 rows for a full round).
